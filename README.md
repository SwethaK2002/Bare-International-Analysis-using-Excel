# Bare International Analysis Project

## Objective
The **Bare International Analysis Project** focuses on analyzing evaluation data from different regions to assess performance trends and identify high-performing zones.  
The project demonstrates the use of **Excel-based analytical tools** for performance grading, regional comparison, and data-driven insights.

---

## Excel File
**File:** `Bare-International-Analysis.xlsx`

- Cleaned and structured the dataset for accurate evaluation.  
- Implemented a **Performance Grading System** using the **IFS function** to categorize employees into *High, Average, Low,* and *Bottom* performers.  
- Created **Pivot Tables** to calculate average evaluation scores by zone and analyze performance distribution.  
- Determined the **region with the highest percentage of High Performers** through calculated fields and percentage analysis.  
- Added **charts and visual dashboards** to highlight performance patterns and regional comparisons.

---

## Insights & Findings
- **Regional Trends:** Identified zones with consistently high or low performance averages.  
- **High Performer Ratio:** Highlighted regions with the highest proportion of high performers as benchmarks.  
- **Performance Shifts:** Observed changes and trends in performance over time for strategic planning.  
- **Category Distribution:** Analyzed the spread of performance categories (*High, Average, Low, Bottom*) to locate improvement areas.
